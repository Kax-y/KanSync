Goals

  • Build a website to import Kanban boards from Jira, Trello, or Github Project.
  • Implement user authentication to allow users to connect their accounts to the aforementioned services
  • Develop connectors for each service to extract data from their respective APIs
  • Standardize the extracted data into a common format.
  • Create a database to store the transformed data.
  • Develop a front-end interface to display the imported Kanban boards.
  • Implement features to generate charts and reports from the imported data, enable collaboration, and possibly integrate Google Calendar
